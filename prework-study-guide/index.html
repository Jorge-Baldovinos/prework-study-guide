HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic HTML plays an important role in the functionality and accessability of a webpage.
  • Metadata, contained within the head element, offers vital information towards the proper performance of the website as well as its accessability and search engine optimization

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheets and it is a programming language that defines how webpage content is presented to users.
  • CSS determines the colors, aesthetics, visual layout, and organization.
  • The three approaches to styling a webpage using CSS are: inline CSS, internal CSS, and external CSS.
  • Inline CSS apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • Internal CSS contain rules that only apply to that webpage.
  • Internal CSS is embeddewd within the head elements.
  • External CSS uses a linked file to apply all the CSS rules to the webpage.
  • External CSS is the prefered styling method as keeping the CSS rules in a separate file makes design changes easier.
  • A CSS Rule consists of a selector and a declaration block. The selector assigns the rules to an element and the declaration block contains the property and value assigned.
